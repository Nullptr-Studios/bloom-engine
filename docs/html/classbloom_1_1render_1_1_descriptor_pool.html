Detailed Description

Manages a pool of descriptor sets.

This class represents a Vulkan descriptor pool, which is responsible for allocating descriptor sets. Descriptor pools are used to allocate a collection of descriptor sets with specific descriptor types and counts. Using a pool is more efficient than allocating individual descriptor sets. This class provides methods for allocating and freeing descriptor sets, as well as resetting the pool.
