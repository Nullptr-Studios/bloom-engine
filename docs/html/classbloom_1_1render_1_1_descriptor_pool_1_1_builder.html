Detailed Description

Helper class to build DescriptorPool objects.

Simplifies the creation of DescriptorPool objects by allowing you to specify pool sizes, flags, and the maximum number of sets using a fluent interface. This builder handles the details of setting up the VkDescriptorPoolCreateInfo struct.
