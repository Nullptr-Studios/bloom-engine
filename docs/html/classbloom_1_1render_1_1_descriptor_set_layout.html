Detailed Description

Defines the layout of descriptors in a shader.

This class represents a Vulkan descriptor set layout, which serves as a blueprint for creating descriptor sets. It specifies the types and bindings of descriptors that can be accessed by a shader. For example, it defines whether a descriptor is a uniform buffer, a sampled image, or another type, and at which binding point in the shader it's accessible. Multiple descriptor sets can be created from the same layout, ensuring consistency between the shader and the data it uses. This layout is crucial for binding resources like buffers and images to the shader pipeline.
