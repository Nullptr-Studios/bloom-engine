Detailed Description

Base class for all objects in the game world.

The Object class provides a common interface and basic functionality for all objects in the game. It manages the object's ID and provides virtual methods for the game loop stages: Begin, Tick, and End. Derived classes can override these methods to implement their specific behavior.
