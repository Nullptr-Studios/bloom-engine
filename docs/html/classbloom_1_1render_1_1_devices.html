Detailed Description

Manages Vulkan devices and related resources.

This class encapsulates the initialization and management of Vulkan devices, instances, surfaces, and other related resources. It provides methods for creating and configuring these resources, as well as helper functions for buffer and image operations. It serves as the central point of interaction with the Vulkan API.
