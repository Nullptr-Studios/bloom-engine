Detailed Description

Contains classes and functions related to rendering in the bloom engine.

The bloom::render namespace encapsulates all the components and utilities required for rendering graphics using Vulkan. This includes managing devices, pipelines, models, shaders, and other rendering-related resources.

The bloom::render namespace is designed to provide a comprehensive set of tools for rendering 3D graphics. It includes classes for managing Vulkan devices, creating and managing graphics pipelines, handling models and their vertex/index buffers, and more. The namespace is structured to facilitate the creation, management, and rendering of 3D objects in a Vulkan-based rendering engine.

Key Classes:

  • Devices: Manages Vulkan devices and related resources.
  • Pipeline: Manages Vulkan graphics pipelines.
  • Model: Represents a 3D model and manages its vertex and index buffers.
  • Renderer: Handles the rendering process, including command buffer management and swap chain recreation.
  • SwapChain: Manages the Vulkan swap chain for rendering.
