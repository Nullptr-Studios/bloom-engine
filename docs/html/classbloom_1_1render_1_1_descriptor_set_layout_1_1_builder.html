Detailed Description

Helper class to build DescriptorSetLayouts.

The Builder class provides a fluent interface for constructing DescriptorSetLayout objects. This approach simplifies the creation process by allowing chained calls to the AddBinding function. It also improves code readability and maintainability by encapsulating the layout creation logic.
