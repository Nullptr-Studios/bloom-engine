Detailed Description

Represents a camera within the game world, providing a view and projection for rendering.

The Camera class inherits from Object and manages the view and projection matrices necessary for rendering the scene from a specific viewpoint. It supports both orthographic and perspective projections. The camera's position and orientation are used to calculate the view matrix, which transforms world coordinates into camera space. The projection matrix then transforms camera space coordinates into clip space for rendering.

Sets an orthographic projection for the camera.

An orthographic projection is a type of projection that does not simulate perspective. Parallel lines remain parallel in the projected image. All parameters of the function describe the view planes in world space.
