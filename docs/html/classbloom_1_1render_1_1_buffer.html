Detailed Description

Represents a Vulkan buffer object.

This class encapsulates a Vulkan buffer and its associated memory, providing methods for data manipulation and interaction with the Vulkan API. It supports storing multiple instances of data within a single buffer, and provides convenience methods for accessing individual instances.

VkDeviceSize bloom::render::Buffer::GetAlignmentSize ( ) const
inline

Gets the alignment size used for instances in the buffer.

This represents the alignment requirement, in bytes, for each instance in the buffer. This value is likely the same as the instance size, but it's important to distinguish between them. It is recommended to review whether a separate alignment member is needed, or if this function should simply return the instance size. This function currently returns the same value as GetInstanceSize().

Returns
The alignment size in bytes.
