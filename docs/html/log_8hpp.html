Detailed Description

Header file for the Log library class.

Author
Xein xgoni.nosp@m.p@gm.nosp@m.ail.c.nosp@m.om
Date
11/21/2024

This file declares the Log class, which provides a centralized logging system for both the engine and the game. The logging functionality is built on top of the spdlog library and supports multiple log levels (e.g., trace, info, warning, error).

The file also includes the necessary macros and definitions for the Bloom and Game logging consoles, ensuring clear separation between engine and game logs.
