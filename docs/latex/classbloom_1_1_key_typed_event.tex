\doxysubsection{Detailed Description}
Represents an event triggered when a printable key is typed. 

The {\ttfamily \doxylink{classbloom_1_1_key_typed_event}{Key\+Typed\+Event}} class provides information about when a key that can be typed (e.\+g., a character or number key) is pressed. This event is used to handle text input.
