\doxysubsection{Detailed Description}
Contains classes and functions related to rendering in the bloom engine. 

The {\ttfamily \doxylink{namespacebloom_1_1render}{bloom\+::render}} namespace encapsulates all the components and utilities required for rendering graphics using Vulkan. This includes managing devices, pipelines, models, shaders, and other rendering-\/related resources.

The {\ttfamily \doxylink{namespacebloom_1_1render}{bloom\+::render}} namespace is designed to provide a comprehensive set of tools for rendering 3D graphics. It includes classes for managing Vulkan devices, creating and managing graphics pipelines, handling models and their vertex/index buffers, and more. The namespace is structured to facilitate the creation, management, and rendering of 3D objects in a Vulkan-\/based rendering engine.
