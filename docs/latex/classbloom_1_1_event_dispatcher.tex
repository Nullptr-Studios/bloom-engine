\doxysubsection{Detailed Description}
A utility class for dispatching events to appropriate handlers. 

The {\ttfamily \doxylink{classbloom_1_1_event_dispatcher}{Event\+Dispatcher}} class provides a mechanism for safely and efficiently handling events by invoking user-\/defined callback functions. It ensures that events are only handled by functions designed for their specific type.
