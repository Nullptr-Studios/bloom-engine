\doxysubsection{Detailed Description}
Helper class to write data to descriptor sets. 

This class simplifies the process of updating descriptor sets with buffer and image information. It provides methods for writing buffer and image data to specific binding points in a descriptor set. The \doxylink{classbloom_1_1render_1_1_descriptor_writer}{Descriptor\+Writer} works in conjunction with a \doxylink{classbloom_1_1render_1_1_descriptor_set_layout}{Descriptor\+Set\+Layout} and a \doxylink{classbloom_1_1render_1_1_descriptor_pool}{Descriptor\+Pool} to manage the descriptor update process. It allows batched updates using vk\+Update\+Descriptor\+Sets for efficiency.
