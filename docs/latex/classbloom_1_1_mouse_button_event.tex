\doxysubsection{Detailed Description}
Base class for mouse button events. 

The {\ttfamily \doxylink{classbloom_1_1_mouse_button_event}{Mouse\+Button\+Event}} class provides functionality for retrieving the button involved in a mouse button event. This class is designed to be extended by specific button event types, such as {\ttfamily \doxylink{classbloom_1_1_mouse_button_pressed_event}{Mouse\+Button\+Pressed\+Event}} and {\ttfamily \doxylink{classbloom_1_1_mouse_button_released_event}{Mouse\+Button\+Released\+Event}}.
