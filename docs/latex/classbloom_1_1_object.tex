\doxysubsection{Detailed Description}
Base class for all objects in the game world. 

The \doxylink{classbloom_1_1_object}{Object} class provides a common interface and basic functionality for all objects in the game. It manages the object\textquotesingle{}s ID and provides virtual methods for the game loop stages\+: {\ttfamily Begin}, {\ttfamily Tick}, and {\ttfamily End}. Derived classes can override these methods to implement their specific behavior.
