\doxysubsection{Detailed Description}
Helper class to build Descriptor\+Set\+Layouts. 

The \doxylink{classbloom_1_1render_1_1_descriptor_set_layout_1_1_builder}{Builder} class provides a fluent interface for constructing \doxylink{classbloom_1_1render_1_1_descriptor_set_layout}{Descriptor\+Set\+Layout} objects. This approach simplifies the creation process by allowing chained calls to the Add\+Binding function. It also improves code readability and maintainability by encapsulating the layout creation logic.
