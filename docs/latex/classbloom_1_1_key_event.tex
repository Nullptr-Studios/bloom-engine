\doxysubsection{Detailed Description}
Base class for all keyboard events. 

The {\ttfamily \doxylink{classbloom_1_1_key_event}{Key\+Event}} class provides the common functionality for keyboard-\/related events, such as retrieving the key code of the event. This class is designed to be extended by specific event types, such as {\ttfamily \doxylink{classbloom_1_1_key_pressed_event}{Key\+Pressed\+Event}} and {\ttfamily \doxylink{classbloom_1_1_key_released_event}{Key\+Released\+Event}}.
