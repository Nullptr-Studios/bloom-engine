\item Classes and Functions use {\ttfamily Pascal\+Case}

\begin{DoxyItemize}
\item NO TABS, 2 spaces
\item Soft line limit of 120 characters
\item Hard line limit of 150 characters
\item Use {\ttfamily \#pragma region} to group functions inside source files if there are a lot of functions
\item Favour longer classes
\item Favour inheritance over ECS
