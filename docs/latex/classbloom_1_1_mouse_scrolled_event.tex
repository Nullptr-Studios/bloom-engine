\doxysubsection{Detailed Description}
Represents an event triggered when the mouse wheel is scrolled. 

The {\ttfamily \doxylink{classbloom_1_1_mouse_scrolled_event}{Mouse\+Scrolled\+Event}} class captures the offset values of the scroll action.
