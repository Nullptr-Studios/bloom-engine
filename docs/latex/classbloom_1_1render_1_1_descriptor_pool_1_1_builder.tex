\doxysubsection{Detailed Description}
Helper class to build \doxylink{classbloom_1_1render_1_1_descriptor_pool}{Descriptor\+Pool} objects. 

Simplifies the creation of \doxylink{classbloom_1_1render_1_1_descriptor_pool}{Descriptor\+Pool} objects by allowing you to specify pool sizes, flags, and the maximum number of sets using a fluent interface. This builder handles the details of setting up the Vk\+Descriptor\+Pool\+Create\+Info struct.
