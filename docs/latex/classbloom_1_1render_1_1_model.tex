\doxysubsection{Detailed Description}
Represents a 3D model in the bloom rendering engine. 

The \doxylink{classbloom_1_1render_1_1_model}{Model} class encapsulates the vertex and index buffers, and provides methods to bind and draw the model using Vulkan command buffers.
