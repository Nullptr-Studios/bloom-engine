\doxysubsection{Detailed Description}
Represents a renderable and potentially interactive object within the game world. 

The \doxylink{classbloom_1_1_actor}{Actor} class extends the \doxylink{classbloom_1_1_object}{Object} class, adding rendering capabilities. It serves as the base class for any object in the game that needs to be visually represented in the world. Actors manage their own textures, materials, and rendering properties. They participate in the game\textquotesingle{}s update loop through the {\ttfamily Begin}, {\ttfamily Tick}, and {\ttfamily End} methods, allowing for dynamic behavior and animation.

Loads textures for the actor\textquotesingle{}s visual representation.

This method loads the textures used to render the actor. It supports loading albedo, roughness-\/metallic-\/occlusion (RMO), normal, and emission textures. If no paths are provided for the textures, placeholder textures will be used instead.


\begin{DoxyParams}{Parameters}

\hline
{\em albedo} & The path to the albedo texture file. The albedo texture defines the base color of the actor\textquotesingle{}s surface. \\
\hline
{\em rmo} & The path to the roughness-\/metallic-\/occlusion (RMO) texture file. This texture controls the surface properties of the actor, including how rough or smooth it appears and how metallic it is. \\
\hline
{\em normal} & The path to the normal map texture file. The normal map modifies the perceived surface normals of the actor, adding detail and depth without increasing the polygon count. \\
\hline
{\em emission} & The path to the emission texture file. The emission texture defines areas of the actor that emit light, creating a glowing effect. \\
