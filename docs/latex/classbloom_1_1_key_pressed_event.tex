\doxysubsection{Detailed Description}
Represents an event triggered when a key is pressed. 

The {\ttfamily \doxylink{classbloom_1_1_key_pressed_event}{Key\+Pressed\+Event}} class provides information about the key being pressed and the number of times the key has been repeated while being held down.
