\doxysubsection{Detailed Description}
Represents a camera within the game world, providing a view and projection for rendering. 

The \doxylink{classbloom_1_1_camera}{Camera} class inherits from \doxylink{classbloom_1_1_object}{Object} and manages the view and projection matrices necessary for rendering the scene from a specific viewpoint. It supports both orthographic and perspective projections. The camera\textquotesingle{}s position and orientation are used to calculate the view matrix, which transforms world coordinates into camera space. The projection matrix then transforms camera space coordinates into clip space for rendering.
