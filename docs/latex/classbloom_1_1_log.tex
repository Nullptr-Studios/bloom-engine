\doxysubsection{Detailed Description}
Centralized logging system for the engine and game. 

The {\ttfamily \doxylink{classbloom_1_1_log}{Log}} class provides a unified interface for handling logging operations within the engine and game. It acts as a wrapper around the {\bfseries{spdlog}} library, offering convenient methods to log messages at various severity levels (e.\+g., trace, info, warning, error, critical).

This class manages two primary logging consoles\+:
\begin{DoxyItemize}
\item {\bfseries{Bloom}} Console\+: Used for logging messages related to the engine\textquotesingle{}s internal systems.
\item {\bfseries{Game}} Console\+: Used for logging messages specific to the game logic and runtime.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The {\ttfamily BLOOM\+\_\+\+API} macro is used to expose this class for external use in a shared library context. 
\end{DoxyNote}
