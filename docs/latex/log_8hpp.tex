\doxysubsection{Detailed Description}
Header file for the Log library class. 

\begin{DoxyAuthor}{Author}
Xein \href{mailto:xgonip@gmail.com}{\texttt{ xgonip@gmail.\+com}} 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
11/21/2024
\end{DoxyDate}


This file declares the {\ttfamily Log} class, which provides a centralized logging system for both the engine and the game. The logging functionality is built on top of the {\bfseries{spdlog}} library and supports multiple log levels (e.\+g., trace, info, warning, error).

The file also includes the necessary macros and definitions for the Bloom and Game logging consoles, ensuring clear separation between engine and game logs.
